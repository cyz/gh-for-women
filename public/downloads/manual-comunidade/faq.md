# Perguntas frequentes (FAQ)

## Preciso saber programar para participar?

Não! A comunidade acolhe pessoas de todos os níveis, inclusive quem está começando agora.

## Os encontros são online ou presenciais?

Temos os dois formatos. Confira a agenda para saber os próximos eventos.

## Como posso contribuir?

Você pode ajudar organizando eventos, criando conteúdo ou revisando materiais como este
manual. Veja o arquivo [contato.md](contato.md) para falar com a organização.
